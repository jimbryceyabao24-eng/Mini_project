In this project, I created a Follow Along Links effect using HTML, CSS, and JavaScript. The main purpose of this project is to make navigation links more interactive by adding a smooth highlight animation that follows the user’s mouse when hovering over each link. Whenever the cursor moves to another link, the highlight automatically changes its position and size to match the hovered item. This project helped me understand how to use event listeners, DOM manipulation, and element measurements such as width, height, and position in JavaScript. Overall, this feature improves the user experience by making the webpage look more dynamic, modern, and professional.
